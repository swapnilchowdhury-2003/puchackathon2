# Premier University Computer Club (PUCC) Official Portal

A comprehensive web platform for PUCC, designed to streamline member registration, event management, and club operations.

## 🚀 Overview
This project was developed during a high-pressure hackathon environment to provide a centralized digital hub for the members of the Premier University Computer Club. It features real-time data management via Firebase and a responsive, dark-themed UI.

## 🛠 Why this architecture?
You will notice the project utilizes **modular HTML files with in-built CSS**. This was a strategic design choice made to facilitate the hackathon requirements:
* **Parallel Development:** With multiple developers on the team, self-contained files allowed us to work simultaneously on different features without merge conflicts.
* **Rapid Deployment:** This modular structure significantly reduced development time and ensured a stable, component-based architecture.
* **Efficiency:** Internal CSS ensured that each page is self-sufficient, making it easier to debug and deploy in a fast-paced environment.

## 💻 Tech Stack
* **Frontend:** HTML5, CSS3, JavaScript (Vanilla).
* **Backend:** Firebase Realtime Database.
* **Architecture:** Component-based modular structure.

### 📦 Project Structure
- `index.html`: Main landing page with club overview.
- `about.html`: History and wing details.
- `wings.html`: Information on 6 specialized wings.
- `events.html`: Event management with real-time filtering.
- `team.html`: Hierarchical team visualization.
- `join.html`: Registration form connected to Firebase.
- `recruiter.html`: Authenticated dashboard for application review.
- `member.html`: Password-protected member dashboard.
