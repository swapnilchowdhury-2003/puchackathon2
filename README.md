# puchackathon2
We're a small team, moving fast, with multiple people editing different pages at the same time. Instead of one shared styles.css, every .html file carries its own inline <style> block. That's on purpose: a shared stylesheet means constant merge conflicts whenever two people touch the same class name, and it means one person's tweak on their page can silently break a page they never opened. Keeping the CSS inline per page lets everyone work on their own page independently and ship it without affecting the rest of the site. The tradeoff is repeated CSS across files — an acceptable cost for a project this size, built this fast, by this many hands.
